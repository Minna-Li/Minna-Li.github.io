Chapter 1：统计学习概论

基础分类

  • 监督学习，非监督学习，强化学习

监督学习

在监督学习中，假设输入输出变量(X, Y)服从联合概率分布 $P(X, Y)$，为什么要有这个假设呢？

  1. $P(X,Y)$可以理解成数据生成的内在机制，现实世界中的每一个样本都是独立同分布的抽样于这个未知但固定的$P(X,Y)$,包括我们的训练集和测试集，只有他们服从相同的分布，在训练集中学到的规律才能泛化应用到测试集的预测中，这是泛化能力的理论前提
  2. 从统计角度来说，$$P(X,Y)=P(X)*P(Y|X),$$ P(Y|X) (已知某个输入数据是X,得到输出数据Y的概率) 是我们最终想学习的目标，它刻画了数据的不确定性来源，比如同一个X,会得到不同的输出Y。
  3. 根据学习目标划分了生成模型和判别模型：
    • 学习联合概率分布 $P(X, Y)$是生成方法；比如朴素贝叶斯、隐马尔可夫模型
    • 直接学习决策函数 $f(X)$ 或者 条件分布$P(Y|X)$是判别方法；比如感知机、决策树、逻辑回归、SVM、k 近邻

无监督学习和强化学习

这里暂时不做讨论

统计学习方法三要素

  • 模型：所要学习的条件概率分布 P(Y∣X)​ 或决策函数 Y=f(X)​ 的集合
  • 策略：从假设空间中选取最优模型的准则或损失函数
  • 算法：根据策略，从模型中求解最优参数的具体计算方法

模型

模型就是假设空间，定义了所有的可能学习结果; 有两种形式：概率模型$P_\theta(X,Y)$ 和 非概率模型（决策函数）$f_\theta(X)$

策略

损失函数L(Y,f(X))

衡量单个样本预测值和实际值的偏差，是非负函数，有几种常用的定义：

  • 0-1损失
  • 平方损失
  • 绝对损失
  • 交叉熵损失： $-\log P(y|x)$, 理论很简单，得到的输出y的概率越接近1，损失就越小（接近0）；当$p_y$越小，损失会急剧增大（趋于无穷），这迫使模型将概率集中到正确类别上，因此尤其应用在分类任务中

风险函数

  • 期望风险 $R_{exp}(f)$:
    这是理论目标，针对模型f的真实分布下的平均：
    $$R_{exp}=\int L(y,f(x))P(x,y) dxdy,$$ 因为P的不可计算性，因为引出了经验风险：

  • 经验风险$R_{emp}(f)$:
    $$R_{emp}(f)=\frac{1}{N}\sum_{i=1}^N L(y_i,f(x_i)),$$
    当N趋于无穷时，根据大数定律可知，$R_{emp}(f)$ → $R_{exp}(f)$.

结构风险最小化（SRM）

是一种策略，如果采用经验风险最小化（ERM）,需要样本量足够大时效果才好，样本量过小容易过拟合，因此给经验风险增加一个惩罚项(正则化项J(f))，构造出结构风险：
$$R_{SRM}(f)=R_{emp}(f)+\lambda J(f)$$

算法

算法就是如何数值化地找到使目标函数（如经验风险或结构风险）最小化的模型参数

常见算法举例：梯度下降法及其变种（SGD、Adam）、牛顿法、拟牛顿法、EM算法

二分类问题

True / False : 指预测是否正确
Positive / Nagative: 指预测的结果（正类/负类）,举例：正类可以理解为分类任务你想找出的群体，比如一个MRI图像识别任务中你想找出的是患病的人，那么患病就是正类，不患病就是负类

  • TP: 正确预测为正类
  • TN: 正确预测为负类
  • FP: 错误预测为正类
  • FN: 错误预测为负类

精确率：在所有我预测为正的样本中，有多少是真的正？

$$Precision=\frac{TP}{TP+FP}$$

  • 预测的准确性

召回率：在所有真实为正的样本中，有多少被我成功找出来了？

$$Recall=\frac{TP}{TP+FN}$$

  • 患病群体我检测出了多少病人
    精确率和召回率通常相互矛盾：提高阈值（更严格），精确率↑，召回率↓；降低阈值（更宽松），召回率↑，精确率↓。
  • 真实的覆盖率

准确率

$$Accuracy=\frac{TP+TN}{TP+TN+FP+FN}$$

  • 问题：若99%是负样本，模型全预测负，准确率也有99%，但毫无用处。

F1分数

$$F1=\frac{2PrecisionRecall}{Precision+Recall}$$

  • 为什么用调和平均？​ 因为它惩罚极端值。只有当精确率和召回率都高时，F1才高。算术平均可能掩盖问题。

假正率：主要用于ROC曲线

在所有真实为负的样本中，有多少被错误地判成了正？
$$FPR=\frac{FP}{FP+TN}$$

  • 意义：衡量“误伤”无辜的比例。

ROC曲线

当阈值变化时，召回率(TPR)​ 和 假正率(FPR)​ 会同时变化。以FPR为横轴，TPR为纵轴，描点连成的曲线就是ROC曲线。

  • ROC图像如果是一条对角线，意味着模型是随机猜测，没学到任何有效特征
  • ROC曲线的折点越靠近左上角代表模型性能越好（在相同的FPR下获得更高的TPR）
  • AUC: 曲线下的面积。越接近1表示模型性能越好。

分类问题 vs. 回归问题

维度 分类问题 回归问题
输出变量Y 离散的类别标签
$Y \in \{c_1, c_2, …, c_K\}$		 连续的实数值
$Y \subseteq \mathbb{R}$ 或 $\mathbb{R}^n$
本质目标 学习决策边界，对输入进行分类 学习映射函数，对输入进行数值预测
学习目标 最小化分类错误，找到类别边界 最小化预测值与真实值的距离
模型输出形式 1. 直接输出类别标签
2. 输出属于各类别的概率		 输出连续的预测值
损失函数 交叉熵、合页损失、0-1损失 均方误差(MSE)、平均绝对误差(MAE)
核心评估指标 精确率、召回率、F1分数、AUC-ROC 均方根误差(RMSE)、平均绝对误差(MAE)、决定系数R²
概率视角 估计条件概率 $P(Y|X)$，Y为离散分布 常假设 $Y|X \sim \mathcal{N}(f(X), \sigma^2)$，为连续分布
决策边界 明确存在，可以是线性或非线性 不存在明确边界，是连续函数曲线/曲面
典型算法 逻辑回归、SVM、决策树、朴素贝叶斯 线性回归、多项式回归、回归树、SVR
应用场景 图像分类、垃圾邮件检测、疾病诊断 房价预测、销量预测、温度预测
  • 根本区别：输出变量Y是离散（分类）还是连续（回归）
  • 核心目标：分类关注“是哪一个”，回归关注“是多少”
  • 概率解释：分类输出可解释为类别概率，回归输出是点估计（可加置信区间）

特殊模型说明

  • 逻辑回归：名为“回归”，实为分类模型，通过Sigmoid函数输出概率
  • SVR/分类树：体现了模型可通过改变损失函数/输出处理适应两类问题

课后习题

1.1

说明伯努利模型的极大似然估计以及贝叶斯估计中的统计学习三要素，并推导贝叶斯估计中最优参数(\hat{\theta}_{\text{Bayes}})的表达式。
（补充：伯努利模型样本满足(x_1,x_2,\dots,x_N \in {0,1})，独立同分布，其概率分布为(P(X=1)=\theta)，(P(X=0)=1-\theta)，(\theta \in [0,1])）

  1. 贝叶斯估计的统计学习三要素

  • 模型（假设空间）：伯努利分布族，即(\mathcal{F}=\left{P_\theta(x)=\theta^x(1-\theta)^{1-x},\ \theta \in [0,1]\right})，其中(x \in {0,1})，(\theta)为待估计参数。

  • 策略：最大化后验概率（本质是结构风险最小化思想），即选择参数(\theta)使得后验概率(P(\theta \mid x_1,x_2,\dots,x_N))最大；最终贝叶斯估计值为后验分布的期望(\hat{\theta}_{\text{Bayes}} = \mathbb{E}[\theta \mid x_1,x_2,\dots,x_N])。

  • 算法：利用贝叶斯定理求解后验分布，通过解析法计算后验期望，得到最优参数估计值。

  1. 最优参数(\hat{\theta}_{\text{Bayes}})推导步骤

为了简化后验分布的计算，选择与伯努利分布共轭的先验分布——Beta分布（共轭分布的优势：后验分布与先验分布同类型，无需复杂积分）。
Beta分布的概率密度函数为：
$$p(\theta) = \text{Beta}(\theta; a,b) = \frac{\Gamma(a+b)}{\Gamma(a)\Gamma(b)} \theta^{a-1}(1-\theta)^{b-1}$$

由贝叶斯定理：$$p(\theta \mid x_1,\dots,x_N) = \frac{p(x_1,\dots,x_N \mid \theta) \cdot p(\theta)}{p(x_1,\dots,x_N)}$$
其中似然函数(p(x_1,\dots,x_N \mid \theta))：由样本独立同分布，得$$p(x_1,\dots,x_N \mid \theta) = \prod_{i=1}^N P(x_i;\theta) = \prod_{i=1}^N \theta^{x_i}(1-\theta)^{1-x_i}$$

因此，后验分布可表示为（正比于分子）：
$p(\theta \mid x_{1:N}) \propto \prod_{i=1}^N \theta^{x_i}(1-\theta)^{1-x_i} \cdot \theta^{a-1}(1-\theta)^{b-1} = \theta^{a-1+\sum_{i=1}^N x_i} (1-\theta)^{b-1+\sum_{i=1}^N (1-x_i)}$

故后验分布为(\theta \mid x_{1:N} \sim \text{Beta}\left( \alpha, \beta \right))，其中(\alpha = a + \sum_{i=1}^N x_i)，(\beta = b + \sum_{i=1}^N (1-x_i))。

Beta分布期望公式：若(\theta \sim \text{Beta}(\alpha, \beta))，则(\mathbb{E}[\theta] = \frac{\alpha}{\alpha + \beta})，代入(\alpha,\beta)：
(\hat{\theta}{\text{Bayes}} = \frac{a + \sum{i=1}^N x_i}{\left(a + \sum x_i\right) + \left(b + \sum (1-x_i)\right)}= \frac{a + \sum_{i=1}^N x_i}{a + b + N})

当(a=1,b=1)（均匀分布先验），贝叶斯估计为(\hat{\theta}{\text{Bayes}} = \frac{1 + \sum x_i}{2 + N})（拉普拉斯平滑）；对比极大似然估计(\hat{\theta}{\text{MLE}} = \frac{1}{N}\sum x_i)，贝叶斯估计引入先验可避免小样本过拟合。
