记住Numpy的设计原则是

  1. 基础、高频功能 → 提升到顶级（np.dot, np.array）
  2. 专业、独立功能 → 作为子包导出（np.linalg, np.fft）
    • linalg : 线性代数，因为用户需要明确知道在使用线性代数
    • fft : 傅里叶变换，专业功能
    • random : 随机数生成，明确的功能域
    • polynomial : 多项式
    • ma : 掩码数组，高级功能
  3. 实现细节 → 完全隐藏（core, compat）

Numpy广播机制

numpy广播对象必须是numpy数组，如果是列表或者元组需要先转为数组！

  • case 1 : 纯列表的加法和乘法是元素拼接和重复！(元组情形类似)
list1 = [1, 2, 3]
list2 = [4, 5, 6]

# 1. 列表相加：拼接，不是广播！
result = list1 + list2
print("列表相加（拼接）:", result)  # [1, 2, 3, 4, 5, 6]

# 2. 列表乘法：重复，不是广播！
result = list1 * 3
print("列表乘以标量（重复）:", result)  # [1, 2, 3, 1, 2, 3, 1, 2, 3]

# 3. 想要逐元素相加？需要手动操作
result = [x + y for x, y in zip(list1, list2)]
print("列表逐元素相加（手动）:", result)  # [5, 7, 9]

广播的本质：数组自动扩展维度并逐元素运算，但所有广播行为必须遵从两条核心规则：

  1. 规则1：维度对齐(从右到左匹配维度)
    将两个数组从最右侧开始比对，满足以下条件之一即可广播：
    (1). 维度大小相等
    比如数组A形状 (2,3),数组B形状 (3,)，最右侧维度大小都为3，可以广播
    比如数组A形状 (2,3),数组B形状 (2,)，最右侧维度大小A为3，B为2(不同且非1)，无法广播
    (2). 其中一个数组的维度大小为1
# 示例各种广播情况
examples = [
    # (数组A形状, 数组B形状, 是否能广播, 结果形状)
    ((3,), (), True, (3,)),          # 标量广播到向量
    ((2,3), (3,), True, (2,3)),      # 行广播到矩阵
    ((2,3), (2,1), True, (2,3)),     # 列广播到矩阵
    ((2,3,4), (3,1), True, (2,3,4)), # 复杂广播
    ((2,3), (2,), False, None),      # 不能广播
]
  1. 规则2： 维度扩展(补1原则)
    若两个数组的维度数不同，先给维度数少的数组往左侧补 1，直到维度数一致，再应用规则1
A = np.array([[1, 2, 3],
              [4, 5, 6]])  # (2, 3)
b = np.array([10, 20])  # (2,)
# result = A + b  # ❌ ValueError

# 解决方案1：reshape为列向量
b_column = b.reshape(-1, 1)  # (2, 1)
result1 = A + b_column  #此时满足广播规则

# 解决方案2：转置A（改变计算逻辑）
A_transposed = A.T  # (3, 2)
result2 = A_transposed + b  # 广播到(3,2)
print("行广播（转置后）:\n", result2.T)  # 再转置回来
