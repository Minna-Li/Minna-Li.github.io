一、Python生态（物理层级）

基本概念：

  • 模块module ：一个.py文件就是一个模块
  • 包package ： 由多个相关的模块组成的目录就是包，包下必须包含__init__.py文件
  • 库library : 多个功能相关的包封装构成一个库，可以直接上传到PyPI（Python Package Index），这是 Python 官方的包仓库
  • 框架framework：提供一个架构，遵循特定的规则写代码

包设计原则:

  1. 扁平优于嵌套（但不要太扁平）
  2. 每个包/模块有单一明确的责任
  3. 在顶层的__init__.py 文件中要做以下事情：
    • 控制公开的API
    • 使用__all__明确导出内容
    • 定义延迟加载提升性能，比如numpy中的__getattr__()
    • 提供清晰的导入路径

常用库及其功能：

数据处理与分析库

  • Numpy
  • Pandas
    数据处理库，基于numpy构建，专门用于结构化数据的分析、处理、清洗、转换、统计
  • SciPy
    科学计算扩展

机器学习核心库

  • Scikit-learn
    传统机器学习一站式解决方案
  • XGBoost / LightGBM / CatBoost
    树模型三巨头，比赛/工业界最爱

面试必问：GBDT原理、特征重要性、调参技巧

深度学习与大模型框架

  • PyTorch
  • Transformers (Hugging Face)
    大模型生态核心，NLP必备

必须掌握：
BERT、GPT、T5等主流架构
Tokenizer的使用
Trainer API
Pipeline快速推理

  • Tensorflow(keras)

可视化和调试库

  • Matplotlib
    基础绘图库
  • Seaborn
    统计可视化，更美观

其他实用工具库

  • tqdm
    进度条

  • Logging
    日志记录

  • Pytest/Unittest
    代码测试框架
