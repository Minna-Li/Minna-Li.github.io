Python编程-Python层级概念
一、Python生态（物理层级）基本概念： 模块module ：一个.py文件就是一个模块 包package ： 由多个相关的模块组成的目录就是包，包下必须包含__init__.py文件 库library : 多个功能相关的包封装构成一个库，可以直接上传到PyPI（Python Package Index），这是 Python 官方的包仓库 框架framework：提供一个架构，遵循特定的规则写代码 包设计原则: 扁平优于嵌套（但不要太扁平） 每个包/模块有单一明确的责任 在顶层的__init__.py 文件中要做以下事情： 控制公开的API 使用__all__明确导出内容 定义延迟加载提升性能，比如numpy中的__getattr__() 提供清晰的导入路径 常用库及其功能：数据处理与分析库NumpyPandas数据操作与预处理引擎，专门用于结构化数据的分析、处理、清洗、转换、统计.
